
# Ellty

## Installation

1. **Clone the repository**

    
    git clone https://github.com/prakhar788/ellty.git
    cd ellty
   

2. **Install dependencies**

   
    npm install
    

3. **Start the development server**

   
    npm run dev
    

4. **Open your browser**

    Visit \`http://localhost:3000\` to see the application running.
